## ðŸš€ Quick Start

### Installation

```bash
npm install w-ui
# atau
yarn add w-ui
# atau
pnpm add w-ui
```

### Setup Tailwind CSS

W-UI menggunakan Tailwind CSS. Update `tailwind.config.js` Anda:

```js
module.exports = {
  content: [
    "./src/**/*.{js,ts,jsx,tsx,mdx}",
    // PENTING: Include path ke w-ui library
    "./node_modules/w-ui/dist/**/*.{js,ts,tsx}",
  ],
  theme: {
    extend: {
      colors: {
        primary: {
          // Custom primary colors
          500: "#3b82f6",
          600: "#2563eb",
        },
      },
    },
  },
  plugins: [],
};
```

### Basic Usage

```tsx
import { Button, Card, CardHeader, CardContent, Input } from "w-ui";

export default function MyPage() {
  return (
    <Card>
      <CardHeader>
        <h2>Welcome to W-UI</h2>
      </CardHeader>
      <CardContent>
        <div className="space-y-4">
          <Input label="Email" type="email" placeholder="Enter your email" />
          <Button variant="primary">Get Started</Button>
        </div>
      </CardContent>
    </Card>
  );
}
```
