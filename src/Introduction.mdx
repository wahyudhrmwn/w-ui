## üöÄ Quick Start

### Installation

```bash
npm install @wahyudhrmwn/w-ui
# atau
yarn add @wahyudhrmwn/w-ui
# atau
pnpm add @wahyudhrmwn/w-ui
```

### ‚ö†Ô∏è PENTING: Import CSS Styling

Setelah menginstall library, Anda **HARUS** import CSS styling-nya:

```tsx
// Di _app.tsx (Pages Router) atau layout.tsx (App Router)
import "@wahyudhrmwn/w-ui/styles"; // ‚Üê WAJIB!

// Import komponen
import { Button, Card, Input } from "@wahyudhrmwn/w-ui";
```

**‚ö†Ô∏è Tanpa import CSS ini, komponen akan tampil tanpa styling!**

### Setup Tailwind CSS

W-UI menggunakan Tailwind CSS. Update `tailwind.config.js` Anda:

```js
module.exports = {
  content: [
    "./src/**/*.{js,ts,jsx,tsx,mdx}",
    // PENTING: Include path ke @wahyudhrmwn/w-ui library
    "./node_modules/@wahyudhrmwn/w-ui/dist/**/*.{js,ts,tsx}",
  ],
  theme: {
    extend: {
      colors: {
        primary: {
          // Custom primary colors
          500: "#3b82f6",
          600: "#2563eb",
        },
      },
    },
  },
  plugins: [],
};
```

### Basic Usage

**1. Import CSS di file utama aplikasi (\_app.tsx atau layout.tsx):**

```tsx
// _app.tsx (Next.js Pages Router)
import "@wahyudhrmwn/w-ui/styles"; // ‚Üê PENTING!

export default function App({ Component, pageProps }) {
  return <Component {...pageProps} />;
}
```

**2. Gunakan komponen di halaman/komponen Anda:**

```tsx
import {
  Button,
  Card,
  CardHeader,
  CardContent,
  Input,
  Badge,
  Alert,
  Avatar,
  Tabs,
} from "@wahyudhrmwn/w-ui";

export default function MyPage() {
  return (
    <Card>
      <CardHeader>
        <div className="flex items-center gap-3">
          <Avatar src="/avatar.jpg" alt="User" size="md" />
          <div>
            <h2>Welcome to W-UI</h2>
            <Badge variant="success">Ready to use</Badge>
          </div>
        </div>
      </CardHeader>
      <CardContent>
        <Alert type="info" title="Informasi">
          Library W-UI menyediakan 50+ komponen siap pakai!
        </Alert>

        <div className="space-y-4 mt-4">
          <Input label="Email" type="email" placeholder="Enter your email" />
          <Button variant="primary">Get Started</Button>
        </div>
      </CardContent>
    </Card>
  );
}
```
